# Message Queues Test

A small little test in Posix Message Queues, using epoll, pthreads and mqueue from libc

## Building

### Dependencies

 * Linux
 * LIBC
 * pthreads

### Fetching

	git clone https://github.com/graytshirt/mqueue.git

### Compiling

	cd mqueue
	make

## See Also

	man mq_overview
	man epoll
